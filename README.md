# Lily-house

A cozy illustrated house scene created using only HTML and CSS positioning techniques.

This project demonstrates practical use of:

- `position: absolute`
- Layering with `z-index`
- Border-based triangles
- `linear-gradient` and `radial-gradient`
- `border-radius`
- `transform: skew`
- Layered depth and composition

🎨 What’s Inside : 

The scene includes:

- A two-layer house structure  
- Triangular roof using CSS borders  
- Windows with decorative flower details  
- A chimney with layered smoke puffs  
- A fruit tree with layered foliage  
- A stepped pathway  
- Grass patches and background elements

🛠️ How It Works : 

All elements are positioned absolutely inside a container (`house-container`).  
Each component is placed using calculated coordinates to maintain balance and symmetry.

No JavaScript. No images. Just HTML and CSS.
